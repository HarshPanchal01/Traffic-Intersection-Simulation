\begin{abstract}
    \noindent
    This project investigates the optimization of a 4-way traffic intersection using a 2D kinematic simulation. The focus is on finding the optimal traffic light cycle lengths based on varied arrival rates generated by a Poisson distribution, to minimize queue lengths and average wait times.
